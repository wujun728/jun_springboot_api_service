# groovyscript

#### Description
Java 中集成groovy脚本执行接口

#### Software Architecture
Spring boot web

#### Installation

1. 引入架包
2. 添加配置注解@EnableGroovyApi
4. POST请求地址：~/groovy?script=groovy代码
